# ZtjPythonPackages
my python package index.

### Get project
```
git clone https://github.com/ztj1993/py-packages.git
git submodule init
git submodule update
```

### Pull sub modules
```
git submodule foreach git pull origin master
```

### Add sub module
```
git submodule add -- https://github.com/ztj1993/py-mysql.git mysql
```
